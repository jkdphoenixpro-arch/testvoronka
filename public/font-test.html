Примеры использования в CSS:

/* SF Pro Display - Обычный текст (по умолчанию) */
.text-regular {
    font-family: 'SF Pro Display';
    font-weight: 400;
}

/* SF Pro Display - Средний текст */
.text-medium {
    font-family: 'SF Pro Display';
    font-weight: 500;
}

/* SF Pro Display - Полужирный */
.text-semibold {
    font-family: 'SF Pro Display';
    font-weight: 600;
}

/* SF Pro Display - Жирный */
.text-bold {
    font-family: 'SF Pro Display';
    font-weight: 700;
}

/* Inter - Экстра жирный */
.text-extrabold {
    font-family: 'Inter';
    font-weight: 800;
}
